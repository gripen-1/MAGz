one_long_night

This is the first 3D game I made in Godot engine.

Since I was new to game making, the game is poorly optimised. I have a school laptop and it runs the game badly... I am going to fix this is the future while making a new game. Also, the loading time between scenes and booting up the game takes time. The game has 2.6 GB even though I shouldn't have.

If anyone knows how to optimise games and how to fix loading times pelase let me know!

The game is a mixture of FNAF games, the mechanics are simmilar to FNAF1 and FNAF2 but it also contains some of my original mechanics (I think)...

There are 5 entities that you need to look out for. Every single on has a special trait and for you to be able to avoid getting jumpscared... Learning the movement and the sound they make is crucial.

The characters and the some of the game mechanics are described in the game under the Info tab.
